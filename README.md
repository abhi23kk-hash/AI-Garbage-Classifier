# ♻️ Garbage Classification using Computer Vision

## 📌 Project Overview
Garbage Classification using Computer Vision is a machine learning project that automatically identifies and classifies different types of waste such as plastic, paper, metal, cardboard, glass, and organic waste using image processing and deep learning techniques.

The system helps improve waste segregation and recycling efficiency by automatically recognizing garbage types from images.

---

## 🎯 Objectives
- Automate waste classification using Computer Vision
- Improve recycling efficiency
- Reduce manual effort in garbage segregation
- Support smart city waste management systems

---

## 🧠 Technologies Used
- Python
- Computer Vision
- Deep Learning
- TensorFlow / Keras
- OpenCV
- NumPy
- Matplotlib

---

## 📂 Dataset
The dataset contains images of different garbage categories such as:
- Plastic
- Paper
- Glass
- Metal
- Cardboard
- Organic Waste

---

## ⚙️ How It Works
1. Collect garbage image dataset
2. Preprocess images
3. Train a CNN model
4. Predict garbage category from image
5. Display classification result
